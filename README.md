# Financial Coach - Landing Page

Sitio web de marketing profesional para la aplicación móvil Financial Coach.

## Características

- **Diseño Responsivo**: Optimizado para desktop, tablet y móvil
- **Rendimiento**: Construido con Next.js para máxima velocidad
- **SEO Optimizado**: Metadatos y estructura semántica
- **Accesibilidad**: Cumple con estándares WCAG
- **Modo Oscuro**: Soporte automático para preferencias del sistema

## Secciones

1. **Hero**: Presentación principal con CTA
2. **Features**: 6 características principales
3. **How It Works**: Proceso de 4 pasos
4. **Testimonials**: Opiniones de usuarios
5. **Pricing**: Planes de suscripción
6. **CTA Final**: Llamada a la acción
7. **Footer**: Enlaces y información

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Build

```bash
npm run build
npm start
```

## Despliegue

Desplegado en Vercel. Los cambios en main se despliegan automáticamente.

## Tecnologías

- Next.js 14
- React 18
- TypeScript
- CSS Modules
