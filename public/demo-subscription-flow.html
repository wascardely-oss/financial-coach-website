🔄 Flujo del Proceso

  • 1. Usuario selecciona un plan
  • 2. Se envía solicitud a /api/create-checkout-session
  • 3. Se recibe sessionId de Stripe
  • 4. Usuario es redirigido a Stripe Checkout
